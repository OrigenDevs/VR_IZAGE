Prueba el juego VR sin ponerte el casco. Ideal para desarrollar, debuggear y experimentar.

🖥️ Modo Escritorio
Sin casco Solo monitor
Flechas del teclado Rotan la cámara
Mouse Controla la mano
Click izquierdo Activa el gatillo/láser

Al pulsar la tecla P cambias entre un modo y otro al instante, sin reiniciar el juego.

💡 Para qué sirve: Puedes probar el láser, el agarre de objetos, el menú "Entendí", y todo el sistema de pasos sin necesidad de ponerte el casco ni conectar ALVR. Agiliza muchísimo el desarrollo.

📄 DesktopInputController.cs

Script nuevo que se encarga de todo:

// En cada frame, si modo escritorio activo: Flechas → rotación de cámara
Mouse XY → rotación de la mano
Click izq. → activa el gatillo
Tecla P → cambia entre modos
⚙️ Flujo completo: Pulsas P → DesktopInputController activa desktopOverride en la mano → VRHandController deja de buscar el mando real y usa la posición/rotación que le envía el mouse → Cuando haces click, activa el gatillo y el láser aparece. Todo el resto del juego funciona exactamente igual.

Montaje en Unity

Paso a paso para ponerlo en marcha

1
Crea un GameObject vacío
En la escena LOBBI, clic derecho en la jerarquía → Create Empty. Nómbralo InputManager.
2
Asigna el script
Arrastra DesktopInputController.cs desde la carpeta CODIGO/VR/ al GameObject InputManager.
3
Asigna referencias
En el Inspector del InputManager, arrastra:
Target Camera → La cámara principal (Main Camera)
Left Hand → El GameObject Left Controller (el que tiene VRHandController)
4
Ajusta (opcional)
Puedes modificar estos valores en el Inspector:
Mouse Sensitivity → 0.8 (qué tan sensible es el mouse)
Camera Rotate Speed → 60 (velocidad de giro con flechas)
Hand Offset → dónde aparece la mano respecto a la cámara
5
¡A probar!
Dale Play. Verás un mensaje blanco: "Presiona P para activar modo escritorio". Pulsa P y el mensaje se volverá verde: "MODO ESCRITORIO". Mueve el mouse y las flechas para probar.
🔍 El Inspector debería verse así:
Desktop Input Controller (Script)
├─ Target Camera: Main Camera (Camera)
├─ Left Hand: Left Controller (VRHandController)
├─ Mouse Sensitivity: 0.8
├─ Camera Rotate Speed: 60
├─ Hand Offset: (0.3, -0.2, 0.5)
└─ Toggle Key: P

Consejos y trucos

Para sacarle el máximo partido al modo escritorio

🧪 Prueba el láser
Con el modo escritorio activo, mueve el mouse sobre los cubos de la escena. Verás que el láser aparece al hacer click y la mano se pone verde si toca un objeto agarrable.
🎯 Apuntado preciso
El mouse controla la rotación de la mano. Para apuntar a algo, primero rota la cámara con las flechas para centrar la escena, luego usa el mouse para apuntar fino.
🔄 Alterna rápido
Si tienes el casco conectado, pulsa P para ir al modo escritorio, haz tus pruebas, y pulsa P otra vez para volver a VR al instante.
📐 Ajusta el offset
Si quieres que la mano aparezca en otra posición respecto a la cámara, modifica Hand Offset. X=izquierda/derecha, Y=arriba/abajo, Z=adelante/atrás.
🐞 Debug fácil
Los errores del láser, del agarre o del sistema de pasos se ven directamente en el Game View. No necesitas ponerte el casco para ver qué falla.
🚀 Sin ALVR
Puedes desarrollar sin tener ALVR encendido ni el Oculus Go conectado. El modo escritorio funciona aunque no haya ningún dispositivo VR conectado al PC.
📌 Flujo recomendado de trabajo:
1. Trabajas en scripts desde Visual Studio
2. Vuelves a Unity y das Play
3. Pulsas P para modo escritorio
4. Pruebas el cambio con mouse y teclado
5. Paras, ajustas, vuelves a probar
6. Cuando funciona, te pones el casco y pruebas en VR
